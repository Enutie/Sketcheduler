# Sketcheduler

A Drawabox warm-up session generator: it knows which parts of the Drawabox curriculum you've finished, and picks a few of their exercises for a short timed warm-up before you draw.

## Language

**Milestone**:
A unit of the Drawabox curriculum (a lesson *or* a challenge) whose completion adds its warmup-eligible exercises to the Warm-up Pool.
_Avoid_: Lesson (as the umbrella term — the 250 Box Challenge is not a lesson; "lesson" is fine for actual lessons in UI copy)

**Exercise**:
A single named Drawabox drill (e.g. Ghosted Lines, Funnels), belonging to exactly one Milestone, with an optional example-page link.
_Avoid_: Warmup (an Exercise is only *used as* a warm-up), drill, task

**Warm-up Pool**:
The set of Exercises currently available for selection — the union of all completed Milestones' exercises.
_Avoid_: Available exercises, exercise list

**Session**:
One timed warm-up: 2–3 Exercises randomly picked from the Warm-up Pool, run against a fixed-duration countdown.
_Avoid_: Warm-up session (redundant in context), workout
